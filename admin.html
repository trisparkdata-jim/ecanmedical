☁️ Configuração de Sincronização na Nuvem

🎯 Como funciona: Configure a integração com JSONBin.io para salvar conteúdo automaticamente. Após configurar, clique em "💾 Salvar na Nuvem" para publicar as alterações instantaneamente!

Obtha em: jsonbin.io (Gratuito)
Crie um novo Bin no JSONBin.io e cole o ID aqui

🔗 Netlify Deploy Hook (Opcional)

Configure para atualizar o site automaticamente após salvar na nuvem.

Netlify > Site settings > Build & deploy > Build hooks
  1. Acesse jsonbin.io e crie uma conta gratuita
  2. Crie um novo Bin (Private) e copie o ID do bin (ex: 65f2xxxxx...)
  3. Vá em "Master Key" e copie sua API Key
  4. Cole o Bin ID e API Key acima e clique em "Salvar Configuração"
  5. Pronto! Agora clique em "💾 Salvar na Nuvem" para publicar!
